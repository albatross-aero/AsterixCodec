Detailed Description

Provides access to the decoded records.

This class contains two vectors:

  • vector<vector<string*>*>*: the external vector collects the decoded Records; the internal vector contains all the names of decoded Elements.
  • vector<vector<ElementValue*>*>*: the external vector collects the decoded Records, the internal vector contains all values of the decoded Elements
  • vector<unsigned char*>*: the vector contains the raw data for each record
  • vector<unsigned int>*: the vector contains the length in byte of each raw record
  • vector<unsigned int>*: the vector contains the category number of each record



To access a complete record information:

  • The user should provide an index 'i' (to select the i-th Record):
    • getCollectionOfRawRecord()[i], getCollectionOfRawRecordLength()[i], getCollectionOfDecodedElementNames()[i], getCollectionOfDecodedElementValues()[i], getCollectionOfRecordCategoryNumber()[i] -> represent information related to a decoded record with its raw value, its category number, the length in byte of the raw value, two collections with the Element names and the Element values
  • The user should provide two indexes 'i'(for record) and 'j'(for element)
    • getDecodedElementNamesOfARecord(i)[j], getDecodedElementValuesOfARecord(i)[j] -> represent the name and the value of a decoded element in a decoded record
