RecordCollection * DecoderInterface::decodingDatablocksFromFile ( )

DECODE - Parse and decode a given number of datablocks (N). The input file is previously stored from the 'setSourceDatastream' method. The method decode N datablocks, after the decoding phase it store the index of the new datablock to read for the next call. If the datablocks in the input file are finished, the methods return nullptr.

Returns
Collection of decoded records
